# 25.10.2 Release Notes

The O3DE 25.10.2 is a maintenance release that resolves a few issues found in the 25.10.1 release:
- Project Manager: Loosened requirements for the Visual Studio version when building new projects. This ensures newer versions of Visual Studio will not trigger a compatibility error.
- Export Project: Asset files for shading and global illumination are properly bundled into the exported package.
